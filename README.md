## ATTOM analyzer

### You can analysis raw data via simple command
#### Copy analyzer.py to the raw data directory & run command
```shell
%python analyzer.py
```

```shell

attom.run_no()
attom.binary2txt()
attom.txt2xls()
attom.txt2png()
```
